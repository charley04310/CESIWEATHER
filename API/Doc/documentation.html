API Cesi Weather Documentation

Date de derniere mise à jour : 20/06/2021

Présentation

L'api Cesi weather est une api REST qui utilise les méthodes CRUD.
Elle permet d'alimenter une base de données avec les données de température et d'humidité mesurées par les capteurs et de restituer les données sur une interface web.
La gestion des capteurs peut également etre faite à partir d'une interface web.
Pour le verbe Get un format json est retourné et pour les verbes Put et Post un format json est attendu

 Relevés
Méthode EndPoint Description Clés attendues dans le json
Get api/index.php?cptId Permet de récuperer tous les relevés pour un capteur donné
Get api/index.php?id=last&cptId Permet de récuperer le dernier relevé pour un capteur donné
Get api/index.php?dateInf&dateSup&cptId Permet de récuperer les relevés entre deux dates pour un capteur donné
Get api/index.php?dateInf&cptId Permet de récuperer les relevés à partir d'une date pour un capteur donné
Post api/index.php Permet d'ajouter un relevé de temperature et d'humidité pour un capteur donné Cpt_Id, Rel_Temp, Rel_Hum



 Capteurs
Méthode EndPoint Description Clés attendues dans le json
Get api/capteur.php?id Permet de récuperer les données d'un capteur
Get api/capteur.php Permet de récuperer les données de tous les capteurs
Post api/capteur.php Permet d'ajouter un capteur Cpt_Latitude, Cpt_Longitude, Cpt_Adresse, Cpt_Activation, Cpt_Caracteristique
Put api/capteur.php?id Permet de mettre à jour les données d'un capteur Cpt_Latitude, Cpt_Longitude, Cpt_Adresse, Cpt_DateInstallation, Cpt_Activation, Cpt_Caracteristique
Delete api/capteur.php?id Permet de supprimer un capteur
